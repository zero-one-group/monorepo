# Input

A form input field with optional password toggle and copy functionality.

## Example

```tsx
import { Input } from '@repo/shared-ui/components'

// Basic input
<Input placeholder="Basic input" />

// Password input
<Input type="password" placeholder="Enter password" />

// With copy button
<Input value="Copyable text" showCopyButton />

// With external copy button
<Input value="Copyable text" showExternalCopyButton />
```
