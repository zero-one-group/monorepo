# Textarea

Displays a form textarea or a component that looks like a textarea.

## Example

```tsx
import { Textarea } from '@repo/shared-ui/components'

// Basic usage
<Textarea placeholder="Type your message here." />

// Disabled state
<Textarea disabled placeholder="This field is disabled" />

// With default value
<Textarea defaultValue="Initial content" />

// Custom number of rows
<Textarea rows={8} placeholder="Taller textarea" />
```
