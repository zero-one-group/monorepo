# Spinner

Rotating loading indicator to provide visual feedback to the user.

## Example

```tsx
import { LoadingSpinner, LazyLoadingSpinner } from '@repo/shared-ui/components'

// Basic spinner
<LoadingSpinner />

// Lazy loading spinner with delay
<LazyLoadingSpinner delay={1000} />

// Custom lazy loading spinner
<LazyLoadingSpinner
  delay={1500}
  element={<LoadingSpinner />}
  fallback={<div>Loading...</div>}
/>
```
