# Progress

Displays an indicator showing the completion progress of a task.

## Example

```tsx
import { Progress } from '@repo/shared-ui/components'

// Basic usage
<Progress value={60} />

// With size variants
<Progress value={60} size="sm" />
<Progress value={60} size="default" />
<Progress value={60} size="lg" />
```
